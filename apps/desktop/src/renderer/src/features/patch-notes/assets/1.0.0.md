# v1.0.0 패치노트

## 주요 기능

### 전적 검색
- 닉네임으로 플레이어 전적 조회
- 솔로·듀오·트리오 랭크 통계 확인
- 최근 매치 히스토리 및 킬·데스·어시스트 상세 정보

### 실험체 분석
- 랭커 트리오 랭크 전적 기반 실험체별 메타 통계
- 무기별 필터로 승률·RP 획득·빌드·교전 통계 분리 조회
- 나와 비교하기: 랭커 풀 대비 내 지표 테이블 + 순위별 RP 그래프

### 시야 분석
- 망원 카메라·정찰 드론·EMP 드론 설치 및 활용 통계
- 게임별 시야 기여 트렌드 시각화

### 아이템 분석
- 실험체별 주요 빌드 및 아이템 조합 통계
- Epic / Legend / Mythic 등급별 필터

### 랭커 데이터
- 매일 자동 수집되는 아시아 서버 상위 랭커 전적
- 수집 상태 실시간 배너 표시

## 기타
- 클럽 멤버 관리 및 클럽 랭킹 조회
- 플레이어 비교 오각형 차트
