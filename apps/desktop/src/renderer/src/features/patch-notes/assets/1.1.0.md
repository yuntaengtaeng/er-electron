# v1.1.0 패치노트

## 새 기능
- 실험체 분석 페이지 추가 — 무기 종류별 픽률·승률·RP, 특성·전술 스킬, 최종 빌드, 교전 통계 제공
- 조합 승률 보기 페이지 추가 — 2인·3인 조합별 TOP3 비율·평균 RP·픽률 분석 (실험체 필터 지원)
- 조합 승률 디테일 패널 추가 — 조합 행 클릭 시 1~8등 분포, 순위별 평균 RP, 캐릭터별 기여 인라인 표시
- 패치노트 페이지 추가 — 홈 화면 버전 뱃지 클릭으로 이동
- 랭커 데이터 CSV 다운로드 기능 추가
- 시야 분석에 랭커 벤치마크 비교 기능 추가
- 게임 상세 패널에 몬스터 킬·크레딧 내역 표시 추가
- 페이즈 교전 분석 기능 추가
- 능력치 비교에 막대 차트 모드 토글 추가

## 개선
- 크롤러: 팀 멤버 데미지·플레이타임 수집 추가 (`damage_to_player`, `damage_from_player`, `play_time`)
- 크롤러: 사망 정보(`death_details`) 수집 추가
- 시야 분석 지표를 플레이타임 기준으로 개선
- 아이템 분석에서 오해를 유발하는 지표 제거

## 버그 수정
- 크롤러 게임 필터 오류 수정 및 429 재시도 로직 추가
- 아이템 분석 재료 가격 누락 수정 (파편 재료 추가)
- 크레딧 출처 표시명 수정
